# 🌓 02. Sleek Dark Mode Toggle

A modern, responsive, and accessible theme-switching interface. This project focuses on implementing robust state management, system preference detection, and defensive programming principles.

## 🚀 Key Technical Highlights

- **Three-State Theme Management:** Implemented Light, Dark, and Auto modes, allowing users to lock their preference or let the site follow their system settings.
- **System Preference Detection:** Leveraged `window.matchMedia` with real-time event listeners to seamlessly synchronize the application with OS-level theme changes.
- **Defensive Programming:** Built resilient fallback logic in the state evaluator to gracefully handle missing, null, or manipulated `localStorage` data.
- **BEM CSS Architecture:** Managed via CSS Variables (Custom Properties) and the Block-Element-Modifier (BEM) naming convention for highly scalable and maintainable styling.
- **Smooth Visual Transitions:** Engineered fluid, hardware-accelerated scale and color transitions for a premium, sleek user experience.
- **Robust Error Handling:** Integrated explicit error throwing using `try...catch` principles to ensure core functions only receive valid parameters.
