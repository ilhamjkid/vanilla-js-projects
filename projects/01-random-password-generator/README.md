# 🔐 01. Random Password Generator

A secure, responsive, and accessible password generation tool. This project focuses on implementing advanced randomization logic and semantic HTML structure.

## 🚀 Key Technical Highlights

- **Fisher-Yates Shuffle Algorithm:** Implemented a mathematically unbiased shuffling logic to ensure every generated password has maximum entropy.
- **Semantic HTML5:** Built using `<main>`, `<section>`, and `<form>` tags to provide a meaningful document structure.
- **Web Accessibility (A11y):** Integrated `role="alert"` and `aria-live` regions for the notification system, ensuring a seamless experience for screen-reader users.
- **Modern JavaScript:** Leveraged ES6+ features including Destructuring Assignment, Default Parameters, and clean error handling with `try...catch`.
- **CSS Architecture:** Managed via CSS Variables (Custom Properties) for scalable and maintainable styling.
